Para usar este plugin, você precisa de:

  • Uma conta ativa no Google Earth Engine: Uma conta GEE registrada e aprovada (fornece acesso a conjuntos de dados, APIs e ferramentas).
  • Um projeto no Google Cloud (e seu ID): Um projeto Google Cloud vinculado à sua conta GEE (gerencia autenticação, habilitação de API e cotas para os serviços do Earth Engine).

Clique aqui para acessar o guia passo a passo.

O token de autenticação será armazenado localmente, permitindo pular o login no navegador em futuras autenticações. Para trocar de conta, clique em 'Redefinir Autenticação'.
